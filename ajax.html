API Fun

Description of API
The API that I selected was the Metropolitan Museum of Art Collection API. This API returns the ID of every single publically available object in the museum. This is a large collection, so I filtered the objects to objects that are just a part of department one. This was still a large collection of data, so I chose to only display the first 27 available IDs.
I found it here: https://metmuseum.github.io/

Options I used for API request
I used a HTTPRequest

Functionality of this API This is a very useful API. The first application of this API is being able to check if a certain art piece is publically available. Because each object has a unique ID, you can also use this API to see which department a particular art piece is in.
